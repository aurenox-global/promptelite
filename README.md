## 📖 Descripción

**Prompt Elite Mastery** es una guía completa de ingeniería de prompts que combina lo mejor de 5 recursos especializados en un solo documento interactivo. ## 🚀 Características Principales

### ⚡ Token Efficiency
- Eliminación de cortesías y preámbulos innecesarios
- Verbos raíz en lugar de construcciones verbosas
- Delimitadores estrictos para separar contexto de instrucciones

### 📐 Estructura XML/Markdown
```
[ROL] → [TAREA] → [INPUT] → [FORMATO] → [LÍMITES] → [EJEMPLO]
```

### 🔒 Sistema Anti-Hallucination
- Restricciones negativas precisas
- Validación de certeza al 95%
- Delimitadores estrictos para datos

### ⛓️ Chain of Thought Controlado
- Razonamiento separado de respuesta final
- Menos tokens de pensamiento, misma calidad
- Tags `<thinking>` y `<answer>` para parseo

## 🧪 Checklist Pre-Envío

Antes de enviar cualquier prompt, verifica estos 8 puntos:

- [ ] ¿La instrucción principal está en las primeras 2 líneas?
- [ ] ¿Especificaste el formato de salida exacto?
- [ ] ¿Eliminaste todo saludo, preámbulo y cortesía?
- [ ] ¿Incluyes al menos 1 restricción de qué NO hacer?
- [ ] ¿Usas delimitadores para separar datos de instrucciones?
- [ ] ¿Tienes solo 1 tarea por prompt?
- [ ] ¿Incluyes un ejemplo si el patrón es no-obvio?
- [ ] ¿El rol asignado es específico (no genérico)?

**Puntuación objetivo:** 7-8/8 ✅

## ⚠️ Anti-Patrones a Evitar

| Error | Costo | Solución |
|-------|-------|----------|
| "Actúa como..." sin contexto | ~40 tokens/iteración | Define rol específico con experiencia |
| "Hazlo mejor" sin especificar | ~150 tokens/ciclo | Di exactamente qué cambiar |
| No limitar longitud de respuesta | ~800 tokens/exceso | Especifica máx N palabras |
| Contexto sin delimitar | ~300 tokens confusión | Usa ###, ---, `<tags>` |
| "Piensa paso a paso" innecesario | ~350 tokens | Solo para matemáticas/lógica compleja |
| Repetir contexto en cada prompt | ~200 tokens/prompt | Usa system prompt fijo |
